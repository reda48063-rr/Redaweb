المكوّنات

  • هيكل HTML5 دلالي مع اتجاه RTL.
  • CSS متجاوب باستخدام Grid وFlex بألوان الجامعة (أخضر/ذهبي).
  • JavaScript (وحدات) للبحث، الفلترة، الترتيب، والمفضلة.
  • بدون أُطر عمل أو Build — جاهز للنشر على GitHub Pages.

طريقة النشر على GitHub Pages

  1. أنشئ مستودعًا جديدًا عامًّا باسم saba-university-portal.
  2. ارفع كل ملفات المشروع إلى جذر المستودع.
  3. من Settings → Pages اختر الفرع main والمسار / (root) ثم حفظ.
  4. سيظهر رابط الموقع بعد قليل.
